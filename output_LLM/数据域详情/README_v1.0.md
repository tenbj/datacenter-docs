## 每个文件包含章节	

1. **域定义** - 该数据域的业务含义和覆盖范围	
2. **业务过程清单** - 该域包含的所有业务过程	
3. **粒度定义** - 每个业务过程的粒度声明和主键组成	
4. **维度清单** - 公共维度和专属维度	
5. **事实表设计** - 事实表列表及类型	
6. **总线矩阵** - 该域的维度矩阵（核心域有）	
7. **典型度量指标** - 常用度量指标
