## 1. 域定义与业务全景	

### 1.1 域定义	

客户域覆盖**客户全生命周期的数据**，包括分群画像、行为分析和复购策略。	

### 1.2 业务范围	

```
客户域核心：客户运营与价值管理
├── 分群画像（客户细分、画像标签）
├── 行为分析（浏览、加购、购买、复购）
└── 复购策略（复购提醒、个性化推荐）
```
