### Mermaid流程图

```mermaid
flowchart LR
    %% 用户入口
    用户((用户))
    
    %% 主流程起始
    用户 --> 创建广告请求[创建广告请求]
    创建广告请求 --> 创建计广告[创建计广告]
    
    %% 创建计广告分支
    创建计广告 --> 填写内容[填写内容]
    创建计广告 --> 编辑信息[编辑信息]
    
    %% 填写内容 - 投放策略分支
    填写内容 --> 投放策略{投放策略}
    投放策略 --> |自动投放| 自动投放[自动投放]
    投放策略 --> |手动投放| 手动投放[手动投放]
    
    %% 自动投放路径
    自动投放 --> 完成设置1[完成设置]
    完成设置1 --> 自动投放2[自动投放<br/>广告产品]
    
    %% 手动投放路径
    手动投放 --> 投放位置{投放位置}
    投放位置 --> |自动筛选| 自动筛选[自动筛选<br/>商家类别]
    投放位置 --> |手动筛选| 手动筛选[手动筛选<br/>空间条件]
    自动筛选 --> 商家设置[选择目标类型<br/>输入关键词]
    手动筛选 --> 筛选设置[填入筛选条件]
    商家设置 --> 完成筛选[完成筛选]
    筛选设置 --> 完成筛选
    完成筛选 --> 选定商品链接[选定商品链接]
    
    %% 填写内容 - 广告模式分支
    填写内容 --> 选择广告模式{选择广告模式}
    选择广告模式 --> |商品| 商品模式[商品]
    选择广告模式 --> |视频| 视频模式[视频]
    
    商品模式 --> 设置广告创意1[设置广告位特性<br/>新增展示方式]
    视频模式 --> 设置广告创意2[设置广告效创意<br/>交互选择模块]
    
    设置广告创意1 --> 设置投放频道[设置投放频道]
    设置广告创意2 --> 设置投放频道
    设置投放频道 --> 验证投放规则[验证投放规则]
    
    %% 编辑信息分支
    编辑信息 --> 授权广告[授权广告]
    授权广告 --> 付费类型{付费类型}
    
    付费类型 --> |VCPM签约化| VCPM[VCPM签约化授权]
    付费类型 --> |CPC按效果| CPC[CPC按效果付费]
    付费类型 --> |CPC竞价| CPC竞价[CPC竞价量]
    
    VCPM --> 广告类型{广告类型}
    CPC --> 广告类型
    CPC竞价 --> 广告类型
    
    %% 广告类型分支
    广告类型 --> |垂类| 垂类[垂类]
    广告类型 --> |通投| 通投[通投]
    
    %% 垂类路径
    垂类 --> 垂类解约{解约方}
    垂类解约 --> |品牌店| 品牌店1[亚马逊品牌推理院店]
    垂类解约 --> |商品详情| 商品详情1[商品详情]
    品牌店1 --> 选择品牌1[选择品牌和节目]
    选择品牌1 --> 商品列表1[商品列表]
    商品列表1 --> 添加产品1[添加推广产品]
    商品详情1 --> 添加产品1
    
    %% 通投路径
    通投 --> 通投解约{解约方}
    通投解约 --> |品牌店| 品牌店2[亚马逊品牌推理院店]
    通投解约 --> |商品详情| 商品详情2[商品详情]
    品牌店2 --> 选择品牌2[选择品牌和节目]
    选择品牌2 --> 商品列表2[商品列表]
    商品列表2 --> 添加产品2[添加推广产品]
    商品详情2 --> 添加产品2
    
    %% 最终汇合
    添加产品1 --> 设定投放规则[设定投放规则]
    添加产品2 --> 设定投放规则
    设定投放规则 --> 投放验据[投放规则验据<br/>销量预测]
```

## 流程说明

### 流程图1：广告创建业务流程

1. **入口流程**: 用户发起创建广告请求
2. **投放策略**: 自动投放/手动投放
3. **广告模式**: 商品模式/视频模式
4. **付费类型**: VCPM/CPC
5. **广告类型**: 垂类/通投

### 流程图2：后台操作流程

1. **广告类型**: SP广告、SB广告、SD广告
2. **SP广告**: 自动投放、手动投放（关键词/商品）
3. **SB广告**: 商品集、焦点模式、视频
4. **SD广告**: 触达、页面访问、转化
5. **活动设置**: 竞价策略、预算设置、否定定向
