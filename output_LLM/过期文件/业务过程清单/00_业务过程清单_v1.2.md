## 识别标准举例说明  

以**交易域 创建订单**为例，说明如何判断一个业务活动是否构成业务过程：  

| 识别标准            | 说明                 | 交易域举例                 |
| ------------------- | -------------------- | -------------------------- |
| 1) 有明确的发生时间 | 事件有确切的时间点   | 订单有明确的下单时间       |
| 2) 有唯一的业务标识 | 事件有唯一ID可识别   | 每个订单有唯一的订单号     |
| 3) 有可度量的数值   | 事件产生可量化的指标 | 订单有订单金额、商品数量等 |
| 4) 可追溯到具体对象 | 能关联到谁/什么/在哪 | 可追溯到用户、店铺、商品等 |

> **简单判断**：能回答谁在什么时间做了什么，产生了多少的事件，就是业务过程。
