## 每个文件包含章节	

1. **域定义** - 该数据域的业务含义和覆盖范围	
2. **业务过程与粒度定义** - 该域包含的业务过程及其粒度声明、主键组成	
3. **维度清单** - 公共维度和专属维度	
4. **事实表设计** - 事实表列表及类型	
5. **总线矩阵** - 该域的维度矩阵（部分域有）	
6. **典型度量指标** - 常用度量指标
