# 数据域修改技能资源目录	

本目录包含数据域修改技能所需的资源文件。	

## 目录结构	

```
data-domain-modifier/
├── SKILL.md              # 技能主说明文件
└── resources/
    └── README.md         # 本文件
```

## 相关文件	

| 文件类型 | 位置 | 说明 |
|----------|------|------|
| 权威流程 | `output_LLM/计划书/数据域流程修正计划书_v1.0.md` | 包含权威的业务流程和仓库分类定义 |
| 数据域定义 | `权威资料/1.1 数据域定义表.md` | 16个数据域的定义 |
| 待修改文档 | `output_LLM/数据域详情/数据域完整梳理_v2.0/` | 数据域详情文档 |

## 使用说明	

1. 使用 `/data-domain-modifier` 命令调用此技能	
2. 或在对话中提及"修改数据域"、"更新数据域文档"等关键词	
3. 技能会自动读取权威资料并按照SKILL.md中的步骤执行修改
